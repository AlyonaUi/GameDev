# GameDev

# Учебный проект - игра "Путешествие Мехрака" #

*Суть игры: Мы играем за умный чемоданчик Мехрак, задача которого собирать потерянные инструменты, чтобы помочь архитектору закончить свой проект*

Мы должны летать и собирать инструменты, появляющиеся на поле, а затем принести их к домику, чтобы, если собрано необходимое количество инструментов, отдать их и победить.

"I" - показать/скрыть инвентарь. Отображает собранное количество инструментов каждого вида.

"Q" - отдать инструменты. Находясь у домика, нужно нажать кнопку (если с первого раза не сработало, но условия все соблюдены, просто нажать ещё раз), тогда игра проверит выполнено ли условие, и если оно выполнено, то покажет окно победы, где можно будет начатьигру заново.

**Открывать в Unity**

Начинать со сцены "MainMenu", а далее "MainMenu -> Game"

**В проекте использовался пакет Zenject и, возможно, его придётся закидывать самостоятельно (без него всё будет плохо).**
